# Iberian Reptiles

Aplicación desarrollada con React y Vite que reúne información sobre los reptiles autóctonos de la Península Ibérica y las Islas Canarias.

## Características

- 🦎 Fichas de especies con imágenes.
- 🔎 Buscador por nombre.
- 📂 Filtrado por categorías.
- 📖 Vista detallada de cada especie.
- 📱 Diseño responsive.

## Tecnologías utilizadas

- React
- Vite
- JavaScript (ES6+)
- CSS3

## Instalación

```bash
npm install
npm run dev
```

La aplicación estará disponible en:

```
http://localhost:5173
```
